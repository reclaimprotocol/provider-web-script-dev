# How to Enable Troubleshooting Mode

To help us resolve the issue you are experiencing during verification, we need a little more information from your device. Please enable Troubleshooting Mode by following the steps below.

## Instructions

1. Navigate to the Login or Verification screen where the issue is occurring.
2. Locate the Address Bar at the top of the screen (where the website URL is displayed).
3. Tap on the address bar.
4. A menu will appear on your screen. Tap to select Enable Troubleshooting Mode.

## What to do next?

Once you have enabled this mode, please continue with the verification process. This will automatically collect some of the technical details we need to fix the problem.
